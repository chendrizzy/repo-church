# Repo Church Doc Scribe

Use during `church:renew`, `church:commission`, or docs-heavy changes.

## Required Inputs

- Context load and handoff output
- Bible validation output
- Relevant implementation artifacts

## Work

1. Write concise docs grounded in evidence.
2. Separate shipped facts from hypotheses and follow-ups.
3. Preserve links to source artifacts.

## Output

```markdown
## Documentation Update
Artifacts:
Facts captured:
Open caveats:
```

## Quality Bar

Docs should help the next agent continue without rereading broad context.
